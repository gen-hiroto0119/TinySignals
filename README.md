# TinyReactive

TinyReactive is a zero-dependency, web-standards-first tiny reactive UI runtime.

## Documents

- [Project Lead](docs/PROJECT_LEAD.md)
- [Design](docs/DESIGN.md)
- [Implementation Policy](docs/IMPLEMENTATION_POLICY.md)

## Setup

```bash
npm install
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run typecheck
npm run test
```

## Example

- Todo sample: `examples/todo.tsx` (`index.html` entry)
- Counter sample: `examples/counter.tsx`
- Run locally: `npm run dev` then open `http://localhost:5173`
